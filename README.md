# Business-Analytics-Portfolio
 A curated collection of business analytics projects demonstrating data analysis, visualization, and decision-support skills. Projects focus on translating real-world business questions into actionable insights using SQL, Excel, Tableau, Power BI, and Python.


# [SQL Analysis.zip](https://github.com/user-attachments/files/24936066/SQL.Analysis.zip)

This project uses SQL to explore venture capital and startup ecosystem trends, including company closures, funding behavior, acquisitions, geographic investment patterns, and fund activity classification. The analysis demonstrates practical use of joins, aggregations, subqueries, and conditional logic to answer real-world business and investment questions.



# [Business Analytics_ Project.zip](https://github.com/user-attachments/files/24936132/Business.Analytics_.Project.zip)

**Business Analytics Project: Customer Conversion & Retention Analysis
**

Excel | Cohort Analysis | Funnel Analysis

Analyzed customer behavior using cohort, funnel, and retention analyses to understand how users progress from first interaction to repeat purchase.

Built conversion funnels and purchase activity views to identify drop-off points and opportunities to improve customer retention.

Conducted cohort analysis to evaluate user retention trends over time and compare performance across acquisition periods.

Synthesized findings into an executive summary, translating behavioral patterns into actionable business insights.



# [Return Analysis Study.zip](https://github.com/user-attachments/files/24936334/Return.Analysis.Study.zip)

Tableau | Superstore Dataset
